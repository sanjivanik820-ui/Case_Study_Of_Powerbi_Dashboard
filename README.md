**🎓 Student Fees Performance Dashboard**

**📌 Project Overview**
The Student Fees Performance Dashboard is an interactive Power BI report designed to analyze student fee collections, pending fees, course-wise performance, and student distribution across different courses and areas. The dashboard helps educational institutions track fee payments and monitor financial performance efficiently.

**📊 Dashboard Highlights**
Total Fees: 21M
Total Paid Fees: 17M
Total Pending Fees: 5M
Course-wise fee collection analysis
Student-wise fee payment tracking
Area and course-based filtering

**🔍 Key Insights**
Tracks total, paid, and pending fees across all students.
Identifies courses with the highest fee collections.
Highlights students with outstanding fee balances.
Analyzes student enrollment distribution by course and city.
Supports better financial planning and fee management.

**🛠️ Tools Used**
Power BI
Microsoft Excel
Data Cleaning & Transformation
Data Visualization

**🎯 Objective**
To provide a comprehensive view of student fee collection performance, helping educational institutions monitor payments, identify pending dues, and make informed financial decisions.

**🚀 Skills Demonstrated**
Power BI • Data Analysis • Dashboard Development • KPI Reporting • Data Visualization • Business Intelligence
