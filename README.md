**🎓 Student Fees Performance Dashboard**

**📌 Project Overview**
The Student Fees Performance Dashboard is an interactive Power BI report designed to analyze student fee collections, pending fees, course-wise performance, and student distribution across different courses and areas. The dashboard helps educational institutions track fee payments and monitor financial performance efficiently.

**📊 Dashboard Highlights**
Total Fees: 21M
Total Paid Fees: 17M
Total Pending Fees: 5M
Course-wise fee collection analysis
Student-wise fee payment tracking
Area and course-based filtering

**🔍 Key Insights**
Tracks total, paid, and pending fees across all students.
Identifies courses with the highest fee collections.
Highlights students with outstanding fee balances.
Analyzes student enrollment distribution by course and city.
Supports better financial planning and fee management.

**🛠️ Tools Used**
Power BI
Microsoft Excel
Data Cleaning & Transformation
Data Visualization

**🎯 Objective**
To provide a comprehensive view of student fee collection performance, helping educational institutions monitor payments, identify pending dues, and make informed financial decisions.

**🚀 Skills Demonstrated**
Power BI • Data Analysis • Dashboard Development • KPI Reporting • Data Visualization • Business Intelligence

**🎓 Course Enrollment and Revenue Dashboard**

**📌 Project Overview**
The Course Enrollment and Revenue Dashboard is an interactive Power BI dashboard designed to analyze course enrollment trends, student participation, and revenue generation across various educational programs. It provides valuable insights into course performance and helps institutions make data-driven decisions.

**📊 Dashboard Highlights**
Total Revenue: 16.7M+
Average Revenue: 2.58M+
Total Students: 64K+
Monthly enrollment trend analysis
Course-wise revenue tracking
Enrollment performance comparison across courses

**🔍 Key Insights**
Full Stack Development courses generated the highest revenue and enrollments.
Enrollment trends help identify peak admission periods.
Course-wise analysis highlights the most popular programs.
Revenue distribution enables better planning and resource allocation.
Interactive filters support detailed exploration of course performance.

**🛠️ Tools Used**
Power BI
Microsoft Excel
Data Cleaning & Transformation
Data Visualization

**🎯 Objective**
To monitor enrollment growth, evaluate course performance, and analyze revenue generation through interactive dashboards and visual reports.

**🚀 Skills Demonstrated**
Power BI • Data Analysis • Dashboard Development • KPI Reporting • Data Visualization • Business Intelligence • Educational Analytics

**🎓 Student Performance & Attendance Dashboard**

**📌 Project Overview**
The Student Performance & Attendance Dashboard is an interactive Power BI dashboard that analyzes student academic performance and attendance records. It provides insights into marks, attendance percentages, subject-wise performance, and overall student progress to support data-driven educational decisions.

**📊 Dashboard Highlights**
Total Marks: 2745
Total Attendance: 3021
Average Marks: 78.43
Student-wise marks analysis
Attendance tracking and monitoring
Subject-wise average performance insights

**🔍 Key Insights**
Identifies top-performing students based on marks and attendance.
Tracks overall attendance trends and student engagement.
Compares performance across different subjects.
Highlights areas where students may need additional support.
Provides a comprehensive view of academic achievement and attendance records.

**🛠️ Tools Used**
Power BI
Microsoft Excel
Data Cleaning & Transformation
Data Visualization

**🎯 Objective**
To monitor student performance, analyze attendance patterns, and provide actionable insights that help improve academic outcomes and student engagement.

**🚀 Skills Demonstrated**
Power BI • Data Analysis • Dashboard Development • KPI Reporting • Data Visualization • Educational Analytics • Business Intelligence
